**Game Title:** Mare Nostrum (working title: Sunlit Isles)

**Emotional Core (one sentence):** You are a Roman exile washed ashore, rebuilding a life from wreckage — every crop planted, NPC befriended, and island colonized is an act of quiet defiance against fate.

**Genre:** Cozy Roman life-sim / survival-crafting / light colony builder

**Tech:** p5.js (CDN), no build step, localStorage saves, browser-native

**The hook that could make this 10/10:** The wreck-to-empire arc — you start with nothing on a barren beach and end commanding a multi-island Roman empire. The emotional journey from exile to Imperator is genuinely compelling IF the early wreck moment lands with weight and the late-game colony feels like payoff.

**What works emotionally:**
- The shipwreck intro with raft-building materials scattered across the beach is strong
- Detailed pixel art NPC sprites (Livia, Marcus, Vesta, Felix) with blink/breathe animations
- Procedural lyre music + ambient waves/birds/crickets via p5.sound
- The sailing cutscene from wreck to home island
- Day/night cycle with solar energy mechanic
- Island expansion (25 levels, Citizen -> Imperator)

**Why:** Documented during initial full codebase audit, March 2026.
**How to apply:** All design decisions should serve the exile-to-empire arc. New features should reinforce this emotional journey, not dilute it.
