**Approved tagline:** "Shipwrecked. Sunlit. Reborn." — three words, Roman economy. Client-supplied.

**Comp line that works:** "If Stardew Valley sailed the Mediterranean and hired a Roman historian as narrative director" — use this in itch.io and press outreach.

**Tone:** Warm but not saccharine. Confident. The game's emotional core is "quiet defiance" — exile rebuilding through patience and craft, not heroics. Copy should feel like the Livia NPC dialogue: specific, earned, not overwrought.

**Avoid:** "embark on a journey", "immerse yourself", "stunning", generic survival-game bullet points.

**Platform strategies confirmed:**
- itch.io: Free-to-play browser game with tip jar (suggested $4.99). Zero friction = discovery. Full 10-chapter arc available — no paywall on content.
- Optional gate: Lock chapters VII-X behind $4.99 if conversion is needed.
- Steam (if listed): $14.99.

**Audio USP framing:** "procedural lyre — every note synthesized live, no audio files" — this is genuinely unusual and technically credible. Lead with it in gamedev-targeted posts.

**Twitter/X tone:** lowercase, spare, poetic. Match the lore tablet voice. No hashtag spam — 3-4 max.

**r/gamedev angle:** Technical credibility. Procedural audio architecture, p5.js single-file browser game, no build step. These are legitimately interesting to that audience.

**Why:** Developed during initial marketing asset creation, March 2026.
**How to apply:** Maintain these decisions across all future copy unless explicitly changed.
