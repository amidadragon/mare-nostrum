**Genre:** Cozy Roman life-sim / survival-crafting / colony builder — browser-native, p5.js, no install.

**Core arc:** Shipwrecked Roman exile → Imperator across 10 chapters. Starts with nothing on a wreck beach. Ends commanding a multi-island colony with an Imperial Bridge connecting the islands.

**Confirmed working systems (safe to market):**
- Wreck beach intro, trireme repair, sailing cutscene to home island
- Farming, fishing, cooking, diving (breath bar, sea creatures, treasure)
- 4 NPCs: Livia (senator's exiled daughter), Marcus (lost-legion veteran), Vesta (crystal prophet), Felix — each with 3-quest personal chains
- 4 companion AI critters (cats, tortoise, crow)
- Colony system on Terra Nova (conquer → colonize → level up → trade routes)
- Imperial Bridge construction (stone causeway with pillars, torches)
- 4 explorable islands: Vulcan (volcanic forge), Hyperborea (frozen ruins + lore tablets), Plenty (spice), Necropolis (underworld, truth about exile)
- 10-chapter main quest chain with dialogue payoffs
- Island levels 1-25: Citizen → Governor → Senator → Consul → Imperator
- Procedural lyre (Dorian/Phrygian/sparse Dorian by game state), ambient layers, SFX — zero audio files
- Day/night cycle, seasons, weather, solar energy
- Animated menu with parallax, god-rays, gold dust

**Stub / broken (do not market yet):**
- 4 new island [E] interactions (visuals work, harvest logic not wired)
- Skill tree UI, pirate raids, fleet system, imperial treasury rank

**Why:** Based on QA agent and CEO agent memory audits, March 2026.
**How to apply:** Only market confirmed-working systems. Don't write copy about the skill tree, pirate raids, or fleet until they ship.
